CSDs of Relatively Long Functions of Real Software


Left column contains a list of functions that were taken from real software. Clicking on the buttons will show you two new columns of the source code and its corresponding CSD. This page is not a tool for building CSD for functions, but rather a collection of interesting functions with their CSD. If you are interested in working with the tool that creates these CSDs and learn about more features it suggests you are invited to visit this link ICSD. What you need is simply copying the source code of the function at hand to the textarea of ICSD then click the convert button.
